화가프롬프트 화풍 전략 확정 — C안 하이브리드 (큐비즘 베이스 + 성격 악센트)

세 가지 접근법 심층 비교 후 C안 확정. 시기 고정(A)은 AI 생성 불안정 + 사용자가 미술사 모름. 공통만(B)은 개인화 부족. 하이브리드(C)가 비주얼 안정 + 개인화 + 바이럴 최적.

결정 1 — C안 하이브리드 채택: 큐비즘 공통 베이스(굵은 윤곽선, 면 분할, 다중 시점, 왜곡 비율)를 모든 그림에 적용. 성격별로 분위기·에너지·색감 악센트만 차별화.
결정 2 — 키워드 방향 전환: 미술사 기법 중심 → 분위기·에너지 중심. 라벨도 "아프리카 미술 시기" → "장난꾸러기 화가" 방향.
결정 3 — 청색 시기 → 블루톤 악센트: 전체 파랑 ❌. 다채로운 색 유지 + 블루 그림자/분위기. melancholy 기질 추가, "dreamy, contemplative mood with deep blue undertones" 방향.

프롬프트 감정 표현 4가지 규칙

장면에 좋아하는/싫어하는 것이 실제로 있을 때와 없을 때의 표현 방식을 구분.

결정: 4가지 규칙 확정 — ① 좋아하는 것 있으면 파란색 강조 ② 싫어하는 것 있으면 회색 흐릿 ③ 없으면 파레이돌리아 ④ 싫어하는 것 없으면 구석에 암시
적용: prompts.ts RULE 1~4로 반영 완료

컨셉프롬프트 파레이돌리아 효과

좋아하는 것을 직접 그리면 흥미가 떨어지고, AI 결과가 정확하지 않을 때 실망할 위험. 대신 자연물이 형태를 닮는 방식으로.

결정: 파레이돌리아 효과 채택 — 좋아하는 강도에 따라 뚜렷함 조절 (보통→색감만, 많이→형태 닮음, 집착→환각수준)
적용: prompts.ts EMOTIONAL EXPRESSION 섹션 반영 완료

컨셉 고양이 시점 전환

"고양이를 그린 그림" → "고양이가 화가라면 이 장면을 이렇게 그릴 거야!" 컨셉 확립.

결정: 결과물은 고양이가 바라보는 장면을 고양이가 직접 그린 그림. 입력에 "장면 설명 + 장면 사진" 추가.
