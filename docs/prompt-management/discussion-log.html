화가프롬프트 피카소 시기별 기법 강화 + 고양이×피카소 연결

현재 프롬프트의 피카소 특징이 추상적이라 AI가 매번 다른 스타일로 그림. 시기별 구체적 기법을 promptKeywords에 반영 필요.

개선 방향내용상태
1. promptKeywords 강화6가지 화풍의 키워드를 실제 피카소 기법으로 대폭 확장승인 대기
2. 피카소 공통 특징 추가굵은 윤곽선, 면 분할, 다중 시점, 왜곡 비율 — 부드럽게 적용승인 대기
3. 고양이×피카소 연결넓은 시야각→다중시점, 야간시력→청색톤, 이색형→제한팔레트승인 대기
4. 청색 시기 추가새 기질 추가 (melancholy/lonely) — 쓸쓸한 고양이용옵션 A/B/C 선택 대기
피카소 시기매핑 기질핵심 기법
청색 시기 (1901-1904)미정 (추가 예정)차가운 청색/녹색, 길쭉한 형태, 슬픔/고독, 얇은 윤곽선
장밋빛 시기 (1904-1906)calm따뜻한 톤, 서커스/광대 모티프, 부드러운 곡선
아프리카 미술 (1907)playful각진 형태, 마스크, 기하학적 단순화, 굵은 윤곽선
분석적 큐비즘 (1909-1912)chaotic파편화된 면, 다중 시점, 격자 구조, 단색조
종합적 큐비즘 (1912-1919)(추가 후보)콜라주, 밝은 색, 질감 대비, 문자 요소

프롬프트 감정 표현 4가지 규칙

장면에 좋아하는/싫어하는 것이 실제로 있을 때와 없을 때의 표현 방식을 구분.

결정: 4가지 규칙 확정 — ① 좋아하는 것 있으면 파란색 강조 ② 싫어하는 것 있으면 회색 흐릿 ③ 없으면 파레이돌리아 ④ 싫어하는 것 없으면 구석에 암시
적용: prompts.ts RULE 1~4로 반영 완료

컨셉프롬프트 파레이돌리아 효과

좋아하는 것을 직접 그리면 흥미가 떨어지고, AI 결과가 정확하지 않을 때 실망할 위험. 대신 자연물이 형태를 닮는 방식으로.

결정: 파레이돌리아 효과 채택 — 좋아하는 강도에 따라 뚜렷함 조절 (보통→색감만, 많이→형태 닮음, 집착→환각수준)
적용: prompts.ts EMOTIONAL EXPRESSION 섹션 반영 완료
