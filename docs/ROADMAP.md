### 재미요소 / 바이럴 기능

#### 1. 고양이-집사 애착 유형 테스트 ⭐ (바이럴: 최상 / 난이도: 낮음)
- 과학 근거: 고양이도 4가지 애착 유형 존재 (안정 65.8% / 불안정 34.2%, Current Biology 2019)
- 간단한 퀴즈 → 애착 유형 판별 → 유형별 아트 스타일 매칭
- "당신과 고양이는 '상호 의존' 관계입니다" → 서로 얽힌 큐비즘 스타일
- 성격 테스트류 콘텐츠 = SNS 바이럴 1순위 포맷

#### 2. "집사가 본 나 vs 고양이가 본 나" 비교 이미지 ⭐ (바이럴: 최상 / 난이도: 중간)
- 원본 사진 vs Picatsso 고양이 시점 결과를 나란히 비교
- 비교 밈 포맷은 검증된 바이럴 구조
- 고양이 이색형 시각의 과학적 차이가 시각적으로 극명하게 드러남

#### 3. 고양이 시점 내레이션/캡션 자동 생성 (바이럴: 상 / 난이도: 낮음)
- 아트워크에 고양이의 도도한 시점으로 자동 캡션 부여
- "집사가 또 3분 늦게 밥을 줬다. 용서할 수 없다."
- "이 인간은 사냥을 못 하지만, 체온이 따뜻하므로 허락한다."
- LLM으로 쉽게 구현 + 의인화 밈의 핵심 요소

#### 4. Feline Five 과학적 성격 분석 (바이럴: 상 / 난이도: 낮음)
- PMC 논문 기반 고양이 성격 빅5: 신경증, 외향성, 지배성, 충동성, 친화성
- 각 축의 점수를 레이더 차트로 시각화
- 성격 조합에 따라 고유한 화풍 자동 매칭 → 과학적 근거 강화
- 기존 PRD의 성격 태그 시스템을 과학적으로 리팩토링

#### 5. "고양이 영토 지도" (바이럴: 상 / 난이도: 중간)
- 집사가 고양이 습관 입력 → 고양이 시점 영토 지도 생성
- "이 소파는 나의 왕좌", "냉장고 앞 = 츄르 성지", "창가 = 순찰 초소"
- 고양이 컬러 필터(파란색-노란색 톤) 적용
- 과학적 근거: 실내 고양이도 페로몬으로 명확한 영역 설정

#### 6. "고양이의 하루" 타임라인 아트 시리즈 (바이럴: 중상 / 난이도: 높음)
- 고양이는 야행성이 아니라 박명박모성(crepuscular) — 새벽/황혼에 가장 활발
- 시간대별 다른 화풍의 아트워크 시리즈 생성
- 새벽 4시: 순찰 (박명 톤) → 오후 3시: 낮잠 (따뜻한 톤) → 저녁: 줌미 타임 (역동적 톤)
- 시리즈 콘텐츠 → 재방문 유도

#### 7. "고양이가 집사에게 주는 선물" (바이럴: 중상 / 난이도: 중간)
- 과학 근거: 사냥감을 가져오는 건 집사를 사냥 못하는 가족으로 보고 가르치려는 모성 행동
- 성격 분석 결과에 따라 고양이가 집사에게 줄 선물 아트워크 생성
- 활발한 성격 → 사냥감 테마 / 점잖은 성격 → 꽃/보석 테마

#### 8. 슬로우 블링크 인터랙션 (바이럴: 중 / 난이도: 중간)
- Sussex 대학 2020년 연구: 슬로우 블링크 = 과학적으로 입증된 고양이의 "사랑해"
- 아트워크에서 터치 시 고양이 눈이 천천히 감기는 미니 애니메이션
- "당신의 고양이가 이 그림을 보며 눈을 천천히 깜빡이고 있어요"

#### 9. 꾹꾹이(Kneading) 지수 = 친밀도 게이지 (바이럴: 중 / 난이도: 낮음)
- 고양이-집사 친밀도를 "꾹꾹이 지수"로 시각화
- 높은 지수 → 부드러운 반복 패턴, 따뜻한 톤
- 낮은 지수 → 경계하는, 날카로운 표현

### 화풍 / 스타일 확장

#### 화가 스타일 확장 시스템
- MVP는 피카소만, 이후 다양한 유명 화가 스타일 추가
- 고흐(소용돌이 붓터치), 모네(빛과 색의 흐름), 뭉크(강렬한 감정), 클림트(장식적 패턴)
- 고양이 성격/관상에 따라 어울리는 화가 자동 매칭 or 집사 선택
- 매칭 기준: 성격? 외모? 집사 선호? → **추후 토의 필요**

#### 커스텀 화풍 시스템
- 사용자가 본인 그림체 샘플 2~5장 업로드 → 화풍 특징 추출 → 해당 화풍으로 생성
- 디자이너/일러스트레이터 타겟, 프리미엄 기능으로 수익화 가능

- 이 자료를 기반으로 컬러 팔레트 확정 예정

- 위의 MVP 범위가 적절한지 확인
- 추가하거나 빼야 할 기능?

### 5. 디자인 방향
- 톤앤매너: **아티스틱 (피카소 무드)** — 확정
- 참고할 디자인/사이트: 추후 탐색 예정

- MVP에서는 피카소만, 이후 고흐/모네/뭉크/클림트 등 확장
