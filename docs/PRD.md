# Picatsso - Product Requirements Document (PRD)

## 1. 제품 개요

**제품명:** Picatsso (피캣소)  
**태그라인:** "당신의 고양이가 직접 그린 그림"  
**컨셉:** 반려묘의 성격과 특징을 AI로 분석하고, 고양이가 실제로 보는 색감으로 피카소 스타일의 예술 작품을 생성하는 웹 서비스

### 핵심 가치 제안
- 반려묘의 개성이 담긴 유일무이한 예술 작품
- 과학적 근거(고양이 시각 특성)에 기반한 색감 표현
- 피카소의 예술적 스타일과 고양이 성격의 창의적 결합

---

## 2. 타겟 사용자

### 주요 타겟
- **반려묘 집사:** 자신의 고양이를 특별한 방식으로 기록하고 싶은 사람
- **SNS 콘텐츠 소비자:** 독특하고 공유할만한 이미지를 찾는 사람
- **반려동물 커뮤니티:** 고양이 관련 굿즈/콘텐츠에 관심 있는 집단

### 사용자 페르소나
| 이름 | 나이 | 특성 | 니즈 |
|------|------|------|------|
| 냥집사 김씨 | 25-35 | SNS 활발, 고양이 사진 자주 공유 | 독특한 고양이 콘텐츠 |
| 아트러버 이씨 | 30-45 | 예술/디자인 관심, 인테리어 중시 | 반려묘 테마 아트워크 |
| 선물러 박씨 | 20-40 | 특별한 선물을 찾는 사람 | 커스텀 고양이 굿즈 |

## 3. 핵심 기능

### 3.1 고양이 프로필 입력 (Input)
- **이미지 업로드:** 반려묘 사진 1~3장 업로드
- **텍스트 설명:** 성격, 습관, 특이사항 등 자유 서술
- **선택형 태그:** 성격 키워드 선택 (엉뚱한, 장난꾸러기, 사나운, 잔잔한, 도도한 등)
- **집사-고양이 관계 입력:** 평상시 어떻게 지내는지, 고양이가 좋아하는/싫어하는 것
- **고양이가 보는 장면:** 그림의 주제가 되는 장면 설명 + 장면 사진(선택). "고양이가 화가라면 이 장면을 이렇게 그릴 거야!" 컨셉
- **UX 가이드:** 이미지+설명을 함께 제공하면 더 정확한 결과를 얻을 수 있음을 안내

### 3.2 고양이 성격 분석 (Analysis)
- **AI 분석 엔진:** 업로드된 사진과 설명을 기반으로 고양이 성격/특징 분석
- **분석 결과 표시:** 
  - 성격 유형 (예: "장난꾸러기 탐험가", "도도한 철학자")
  - 성격 키워드 3~5개
  - 매칭된 화풍 스타일 설명
- **사용 AI:** Gemini API (1순위, 이미지 생성과 통합) / Claude API / GPT API (어댑터 패턴으로 교체 가능)

### 3.3 아트워크 생성 (Generation)
- **색감 기반:** 고양이 시각 특성 기반 컬러 팔레트 적용
  - 고양이는 주로 파란색/보라색, 노란색/녹색 계열 인식
  - 빨간색은 흐릿한 갈색/회색으로 변환
  - 이 제한된 색감이 오히려 독특한 예술적 특성으로 작용
- **감정 색감 시스템 (MVP):**
  - 고양이가 좋아하는 대상(츄르, 간식, 친한 집사 등) → 파란색 계열로 선명하게
  - 무관심/싫어하는 대상 → 회색/갈색으로 흐릿하게
  - 집사와의 관계도 AI가 분석하여 반영 (친밀도에 따라 집사를 파랗게 or 회색으로)
  - 재미 포인트: "우리 고양이 눈에 집사는 이렇게 보여요!"
- **화풍 매핑:**
  | 성격 | 화풍 스타일 | 특징 |
  |------|------------|------|
  | 엉뚱한 | 입체파(큐비즘) 강조 | 왜곡된 형태, 다중 시점 |
  | 장난꾸러기 | 다이내믹 표현주의 | 역동적 붓터치, 밝은 톤 |
  | 사나운/싸움꾼 | 강렬한 야수파 | 굵은 선, 강한 대비 |
  | 잔잔한/포근한 | 부드러운 장미빛 시기 | 온화한 곡선, 따뜻한 톤 |
  | 도도한 | 클래식 초상화풍 | 정제된 구도, 우아한 선 |
  | 호기심 많은 | 초현실주의 혼합 | 몽환적, 다양한 오브제 |
- **이미지 생성 엔진:** Gemini Image API (Nano Banana는 Gemini 이미지 모델의 코드네임)
- **결과물:** 1회 생성 시 2~3개 시안 제공

### 3.4 결과물 제공 (Output)
- **미리보기:** 생성된 이미지 미리보기
- **다운로드:** 이미지 다운로드
- **추후 확장 예정:** 워터마크 시스템, SNS 공유, 갤러리/커뮤니티 (ROADMAP Phase 3 참조)


## 5. 기술 아키텍처 (초안)

### 프론트엔드
- **프레임워크:** Next.js App Router (TypeScript, SSR/SEO 지원)
- **스타일링:** Tailwind CSS + shadcn/ui
- **상태관리:** Zustand (슬라이스 패턴으로 모듈 분리)

### 백엔드
- **API 서버:** Next.js API Routes (풀스택 단일 프로젝트)
- **DB:** MVP에서는 미사용 (stateless). 추후 Supabase 도입 예정 (ROADMAP 참조)

### 외부 API
- **고양이 분석 + 이미지 생성:** Gemini API (통합, 하나의 API 키)
  - 분석: Gemini 텍스트/비전 모델
  - 이미지 생성: Gemini Image 모델 (Nano Banana = 코드네임)
  - 대안 (어댑터 패턴으로 교체 가능): Claude API, GPT API

### 인프라
- **호스팅:** Vercel (Next.js 최적화)
- **CDN:** Vercel Edge Network

## 6. 고양이 시각 과학 기반

> ⚠️ 이 섹션은 사용자가 제공할 참고 사이트 자료로 보완 예정

### 알려진 사실 (기본 베이스)
- 고양이는 이색형 색각(dichromatic vision) 보유
- 파란색-보라색 계열과 노란색-녹색 계열을 주로 인식
- 빨간색-주황색 계열은 흐릿한 갈색/회색으로 인식
- 인간보다 야간 시력이 6~8배 뛰어남
- 전체적으로 채도가 낮고 파스텔톤에 가까운 세계

### 컬러 팔레트 변환 로직 (구현 예정)
- 입력 이미지의 색상을 고양이 시각 기반 컬러 스펙트럼으로 재매핑
- 이 변환된 팔레트를 Nano Banana 프롬프트에 반영

## 7. 수익화 전략

### 수익 모델 방향
- **프리미엄(Freemium):** 무료 체험(일일 제한) + 유료 업그레이드
- **광고:** 무료 사용자 대상 배너 광고 (Google AdSense)
- **구독:** 월정액 무제한 생성 + 워터마크 제거 + 고해상도
- **크레딧:** 건별 구매 옵션
- **굿즈:** 실물 상품 연동 (POD 서비스 — 캔버스 프린트, 폰케이스, 머그컵 등)

> 📌 단계별 실행 계획은 `ROADMAP.md` Phase 3 참조
